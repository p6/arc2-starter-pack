I tried to create the store you requested - now let's see if it works. If you have already created a SPARQL endpoint just past the following snippet into a file, save it as sparql.php, replace the default values with your own values and visit the page in your browser:

				<?php

				include_once("../arc/ARC2.php"); // the path to the ARC2.php file

				$config = array(
				  'db_host' => 'localhost',
				  'db_name' => 'arc2',
				  'db_user' => 'root',
				  'db_pwd' => 'root',
				  'store_name' => 'sandbox'
				);

				/* instantiation */
				$ep = ARC2::getStoreEndpoint($config);

				/* request handling */
				$ep->go();
				?>

Congrats! You have successfully set up ARC2 and can use it now.
